HUDdle API

Are you a developer of a navigation app?

Use the HUDdle API to allow your users to access a head-up display

Email support@aquaapps.co to add your app to our "Suported Apps" list

Download Huddle API (45 KB)

Example


                Intent huddleIntent = new Intent(HuddleApi.HUDDLE_API_INTENT);
                huddleIntent.putExtra(HuddleApi.KEY_ETA_TIME, "8:52"); // Always use 12 hour time
                huddleIntent.putExtra(HuddleApi.KEY_ETA_AMPM, "pm"); // "am" or "pm"
                huddleIntent.putExtra(HuddleApi.KEY_DISTANCE, 80); // Always in metres
                huddleIntent.putExtra(HuddleApi.KEY_REMAINING_DISTANCE, 1000); // Always in metres
                huddleIntent.putExtra(HuddleApi.KEY_REMAINING_TIME, 60); // Always in seconds
                huddleIntent.putExtra(HuddleApi.KEY_ICON, HuddleApi.DIRECTION_ROUNDABOUT_6_FLIPPED);
                sendBroadcast(huddleIntent);
